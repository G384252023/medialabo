    各授業回の学習事項

    第1回 WWW と HTML

    今回はHTMLについて学びました。HTML要素とはHTMLが要素から作られるものであり、 <p>や<strong>などといったタグを使って文章を作り、基本となるサイトを作れるようになった。 また、hrefを使ってリンク先のURLを入れることで、他のサイトに飛ばすこともできるようになった。

    演習の成果:演習1-4: HTMLのさまざまな要素

    第2回 CSS による Web ページの装飾

    今回は、CSSについて学びました。 CSSはWebページの装飾のために使うためのもので、CSSによって見た目（文字の太さや大きさ、色など）を変更できるということを学んだ。 HTMLで使うときはheadの中にrelとhrefを使って入れる。 どのような色使いや見た目にすれば見やすくなるかということに苦戦しました。

    演習の成果:練習問題2-3ウェブブラウザ説明ページの装飾

    第3回 JavaScript の基本

    今回はJavaScriptについて学んだ。

    JavaScriptを使ってプログラムを組む。java言語とは似ているようでintやletなど違う点がかなりあるため注意したい。 仮のサイトからコンソールを使ってエラーを吐かないか試しながらプログラムを組むことで、エラーを吐いた時にわかりやすくなる。

    演習の成果:練習問題3-6オブジェクトの参照

    第4回 DOM と JavaScript

    今回はDOMについて学んだ。

    演習の成果:練習問題4-2~4-5説明文の追加、写真表を作る、箇条書きの削除、箇条書きの追加

    第5回 イベント処理

    演習の成果:練習問題5-3イベントを選択する

    苦労したところ 色の設定をする際に、濃い色にし過ぎると文字が見づらく、見やすい色を多用しようとすると同じような色ばかり使ってしまった。
