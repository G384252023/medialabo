    各授業回の学習事項

    第1回 WWW と HTML

    HTML要素とはHTMLが要素から作られるものであり、 <p>や<strong>などといったタグを使って文章を作る。

    演習の成果:演習1-4: HTMLのさまざまな要素

    第2回 CSS による Web ページの装飾

    CSSについて学んだ。 CSSはWebページの装飾のために使う。 CSSによって見た目（文字の太さや大きさ、色など）を変更する方法。HTMLで使うときはheadの中にrelとhrefを使って入れる。

    演習の成果:練習問題2-3ウェブブラウザ説明ページの装飾

    第3回 JavaScript の基本

    今回はJavaScriptについて学んだ。 JavaScriptを使ってプログラムを組む。

    演習の成果:練習問題3-6オブジェクトの参照

    第4回 DOM と JavaScript

    今回はDOMについて学んだ。

    演習の成果:練習問題4-2~4-5説明文の追加、写真表を作る、箇条書きの削除、箇条書きの追加

    第5回 イベント処理
